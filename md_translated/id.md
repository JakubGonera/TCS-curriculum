## Course content

| No. | Course content | Learning outcomes for the course |
| --- | --- | --- |
| 1. | 1. Characteristics of relational databases.    2. Modeling conceptual schemas and implementation schemas in the relational model.    3. Entity relationship model and its transformation to the relational model.    4. Data organization in modern database systems.    5. Characteristics, purpose and techniques of concurrent transaction execution.    6. Database recovery methods after failure.    7. Normalization of relational databases.    8. Structured Query Language SQL.    9. Query optimization.    10. Data warehouses, Big Data and noSQL. | W1,   U1,   K1 |
